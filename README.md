# weberr-studio

Site statique pour Weberr Studio (HTML/CSS/JS + Sanity CMS)

## But
- Front-end statique avec `index.html`, `catalog-sites.html`, `catalog-apps.html`, `style.css`, `main.js`.
- Back-office Sanity pour gérer : vitrines (sitePortfolio) et apps (appSoftware).

## Installation rapide
1. `npm install`
2. `npx sanity start` (dans `sanity-studio`)

## Déploiement
- GitHub → Netlify
